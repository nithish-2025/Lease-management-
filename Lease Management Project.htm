How to Complete Project ?

Step-1: Access Guided project to Complete the Project

Refer Guided Project steps and start working on the milestones & activities with the necessary references provided and develop the project.

Step-2: Update your project status in the workspace

Access Project workspace and update the status of your project using the kanban board available. Move interactive activity tiles for review once the activity is completed. Interact with your mentor directly by commenting on the activity tile.

Step-3: Submit the Project & Get project completion certificate.

Once all the activities of your project are completed, submit fully completed project files into Github through project workspace. The mentor reviews your work and issues certificate.

Instructions:
  1. Explore the guided Project steps provided to complete the project activities
  2. Join the mentor slack channel to interact with mentor (link will be mailed to you)
  3. Update the project completion status regularly through Kanban board
  4. Ensure that all project deliverables are uploaded to the GitHub repository for mentor review.
  5. Project completion certificate will be awarded only if your submitted project files are correct and executable.
